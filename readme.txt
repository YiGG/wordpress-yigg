The mighty YiGG-Button for your WordPress blog

== Description ==

YiGG is a Social News Community focusing on German speaking users. The interface is similar to that of Digg. The YiGG-Button helps to put your news on the homepage of YiGG.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
